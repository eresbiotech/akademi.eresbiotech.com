Hikaye başlıyor: Mira'nın Debug Defteri

Biyoinformatiğe başlamak istiyor ama nereden başlayacağını bilmiyorsan, Mira'nın hata mesajları sana tanıdık gelebilir.

"Ben Mira. Doktoraya başlayacağım. Sanırım. Python, R, Linux, veri setleri ve hayatın bilinmeyen değişkenleriyle aram biraz karışık. Her Çarşamba Instagram'da hatalarımı debug edeceğim. Ama anlatmadıklarımı gizli debug defterimde tutuyorum."

Mira'nın Debug Defteri; biyoinformatiğe başlamak isteyenler için hikayeli kod kartları, küçük debug notları ve Instagram'da devam eden Debug Çarşambası evrenine açılan bir giriş kapısıdır.

Mira'nın Debug Defteri'ni keşfet Ders seçmeye hazır değilsen önce Mira'nın hikayesinden başlayabilirsin.
~/mira/gizli-debug-defteri/init.py
mira = {
  "hedef": "biyoinformatik öğrenmek",
  "durum": "biraz panik, biraz merak",
  "hata_mesaji": "NeredenBaslayacagimHatasi"
}

print("Kod çalışmadı. Hikaye devam ediyor.")
